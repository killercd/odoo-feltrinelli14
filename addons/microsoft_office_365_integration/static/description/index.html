ODOO OFFICE365 Connector

Odoo is an Enterprise Resource Planning(ERP) system which includes CRM, Inventory, Warehouse etc. Microsoft is well known software company who develeops computer Operating System and other related services. One of its best product is Microsoft Office365. Odoo integration with Microsoft Office365 makes it easy to sync the data in Apps i.e Contacts, Outlook, Calendar, OneDrive, Tasks both ways.

  • Installation
  • How its work

Installation

  • Download ODOO OFFICE365 Connector module and place into addons directory
  • Restart Odoo server / service
  • Open setting, activate Developer Mode
  • Go to Apps
  • Update Apps List from top menu
  • Search the ODOO Office365 and install it
  • Happy to use Microsoft Office365


How Its Work


  • Login to Microsoft Account
  • Go to Microsoft Azure Portal
  • Click on Azure Active Directory

  • Create New Application
  • Go to App Registration Menu option
  • Click on Register an Application button

  • Fill Application name
  • Select Supported account type
  • Click on Register button

  • Select Application(Client) ID
  • Click on Add Redirect URIs option

  • Create Redirect URI, which will redirect back after authorization
  • Fill in the Redirect URI like [ODOO-URL]/office_success
  • Click on Configure button

  • Go to Certificate & Secrets
  • Click new Client Secret option
  • Fill in the Client Secret description in pop-up
  • Select expiry limit for Client Secret
  • Click to Add button


  • Install odoo_office365 Module
  • From left menu open Office365 Module
  • Fill in the office365 credentials
  • Click on Connect button
  • It will open an authorization page
  • After authentication, it will redirect to Redirect-URI(ODOO page).

  • Open Microsoft Office365
  • Select Connector menu
  • It support both Import / Export options
  • A datetime filter included always
  • Data sync according to applied filter
  • Note: OneDrive option is available in Contact details

  • During contact create/edit, attachments can sync with OneDrive App
  • Export file button sycn attachments from ODOO to OneDrive App
  • Import files button sync attachments from OneDrive App to ODOO.

  • Import History menu shows how many records are imported
  • There will be no record if nothing is imported.

  • Export History menu show how many records are exported
  • There will be no record if nothing is exported.
